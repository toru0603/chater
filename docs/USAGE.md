# cheter 使い方ガイド

このドキュメントでは、`cheter` をブラウザで使うときの基本的な流れをまとめます。

## 事前準備

1. 開発環境では README の手順でサーバを起動します。
2. ブラウザで `http://localhost:8000` を開きます。
3. 初回接続時は、ブラウザのカメラとマイク利用を許可します。

## 基本的な使い方

1. **表示名** に相手へ見せる名前を入力します。未入力の場合は `Guest` になります。
2. **待ち受け番号** に同じ番号を入力した相手と接続します。
3. **参加** を押すと、カメラとマイクを有効にして待機または接続を開始します。
4. 先に入室した人は待機状態になり、同じ待ち受け番号の参加者が入ると通話が始まります。
5. **退出** を押すと通話を終了して切断します。

## 画面の見方

| 項目 | 内容 |
| --- | --- |
| 表示名 | 自分の表示名です |
| 待ち受け番号 | マッチングに使う共有番号です |
| 参加 | ルームへ参加して接続を開始します |
| 退出 | 通話を終了してルームから抜けます |
| メッセージ | 送信したテキストが画面上に流れます |
| ステータス表示 | 接続状況や待機中メッセージが表示されます |
| 自分 / 相手の動画 | 接続中の映像が表示されます |

## 利用の流れ

### 1. 1対1で使う場合

1. 2人で同じ待ち受け番号を決めます。
2. それぞれのブラウザで `cheter` を開きます。
3. 各自が表示名を入力し、同じ待ち受け番号を入力します。
4. どちらも **参加** を押すと接続されます。

### 2. メッセージを送る場合

1. **メッセージ** 欄に文字を入力します。
2. **送信** を押すか Enter キーで送信します。
3. 送信した内容は画面上に流れるコメントとして表示されます。

## うまくつながらないとき

- **「待ち受け番号を入力してください」** と表示される  
  待ち受け番号を入力してから参加してください。

- **カメラ/マイクを取得できません** と表示される  
  ブラウザの権限設定でカメラとマイクを許可してください。

- 相手とつながらない  
  2人が同じ待ち受け番号を入力しているか確認してください。

- 画面が閉じた、または通信が切れた  
  もう一度ページを開き直して参加し直してください。
